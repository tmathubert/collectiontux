Application web de collection de cartes TUX

On peut gagner des points et des cartes TUX en faisant de bonnes choses à MiNET !
Cette application permet de montrer les cartes TUX qu'on a !

Normalement, on peut acheter une carte TUX avec des points TUX : comme l'application ne gèrera pas les points, on affichera seulement le prix !

CarteTux :

 - type : Local, Matos, Mascotte, Prez, 
 - desc : Description de la carte (ça peut être une carte Local, Matos, ...)
 - prix : prix en points TUX
 - date : date d'obtention de la carte
